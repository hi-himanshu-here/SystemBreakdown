# 🚀 ZeroToScale: 30 Days of Backend & System Design

> Building production-grade backend systems from scratch — one day at a time.

---

## 🧠 About This Project

This repository is my journey to mastering **backend engineering and system design** by building **30 real-world projects in 30 days**.

Instead of just learning theory, I am focusing on:
- Breaking down complex systems into smaller components
- Designing scalable architectures
- Writing production-ready code
- Understanding how systems behave at scale (10 → 1M users)

---

## ⚡ What Makes This Different?

Most tutorials stop at "it works".

This project focuses on:
- 🔥 Scalability
- 🔥 Reliability
- 🔥 System thinking
- 🔥 Real-world constraints

Every project answers:
- What happens at scale?
- Where does it break?
- How do we fix it?

---

## 🛠️ Tech Stack

- **Backend:** Node.js / Python (FastAPI / Django)
- **Database:** PostgreSQL
- **Cache:** Redis
- **Queues:** Kafka / RabbitMQ
- **Realtime:** WebSockets
- **Cloud:** AWS (EC2, S3, RDS)
- **DevOps:** Docker (later stages)
